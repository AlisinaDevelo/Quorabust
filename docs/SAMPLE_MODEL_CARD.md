## Intended Use

Scores pairs of short natural-language questions and returns the probability that the pair is semantically duplicate. Use it as a ranking or moderation signal, not as a sole automated decision system.

## Serving Contract

`POST /predict` accepts `question1` and `question2` arrays of equal length and returns `proba_duplicate` in the same order. `GET /metrics` exposes Prometheus counters and latency histograms.

## Caveats

Performance depends on the training data distribution and threshold. Re-run this card on a current holdout set before comparing artifacts.

## Reproduce

```bash
pip install -e ".[dev]"
python scripts/make_sample_data.py --out data/sample/synthetic_pairs.csv --rows 2000 --seed 42
quorabust-train  --csv data/sample/synthetic_pairs.csv --out models/sample/synthetic_model.pkl --eval-fraction 0.2 --seed 42
quorabust-report --model models/sample/synthetic_model.pkl --eval-csv data/sample/synthetic_pairs.csv --artifact-label synthetic-tfidf-demo.pkl --out docs/SAMPLE_MODEL_CARD.md
```

For a real model card, run the same `train`/`report` commands against the Kaggle Quora Question Pairs CSV instead of the synthetic file.
